# BJS / NCVS documentation (citation notes)

This folder contains Bureau of Justice Statistics (BJS) documentation used to ground descriptions of NCVS data structure and variance estimation.

Included:
- `NCVS_Select_person_level_codebook.pdf` — NCVS Select personal population + personal victimization codebook (legacy instrument; curated variables; overview of NCVS).
- `gvf_users_guide.pdf` — Users’ guide to NCVS Generalized Variance Functions (GVFs).
- `ncvs_variance_user_guide_11.06.14.pdf` — User’s guide to NCVS Direct Variance Estimation.
